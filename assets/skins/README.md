# Bundled skins

Drop a skin folder here to ship it with the app:

```
assets/skins/<name>/
├── po_body.png
└── po_hand.png
```

It appears in **Tray → 設定… → 造型**. User-supplied skins live in the per-user
config dir instead — see [../../docs/skins.md](../../docs/skins.md) for the full
format and locations.
